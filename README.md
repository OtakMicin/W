## IDE FROJECT
 Aplikasi yang membantu user untuk : 
- Mengecek kondisi kesehatan awal (self-check)
- Booking konsultasi dokter
- Mendapatkan informasi obat & kesehatan
- Monitoring kesehatan harian

Tujuan Aplikasi :
- Mempermudah akses layanan kesehatan
- Mengurangi antrian rumah sakit
- Edukasi kesehatan masyarakat

## FITUR & TIMELINE 
**User & Account**
1. Register & Login user
2. Edit profil pengguna
3. Riwayat aktivitas kesehatan

**Kesehatan**
4. Self-check gejala (AI sederhana berbasis input)
5. Rekomendasi tindakan awal
6. Tracking kesehatan harian (makan, tidur, olahraga)

**Konsultasi**
7. Booking dokter
8. Chat dengan dokter

**Obat & Informasi**
9.Database obat
10. Informasi penyakit umum
11. Reminder minum obat

**Lokasi & Akses**
12. Pencarian rumah sakit terdekat
13. Maps lokasi fasilitas kesehatan

**Sistem**
14. Notifikasi (jadwal, obat, konsultasi)
15. Rating & review dokter
16. Admin dashboard

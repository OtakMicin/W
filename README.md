## IDE PROJECT
 Aplikasi yang membantu user untuk :
- Mengecek kondisi kesehatan awal (self-check)
- Booking konsultasi dokter
- Mendapatkan informasi obat & kesehatan
- Monitoring kesehatan harian

Tujuan Aplikasi :
- Mempermudah akses layanan kesehatan
- Mengurangi antrian rumah sakit
- Edukasi kesehatan masyarakat

## FITUR & TIMELINE 
**USER & ACCOUNT**

1. Login user (Login bagian awal web saat dibuka user diminta login dengan email dan password_) - Login Page
2. Edit profil pengguna (Setelah login ada header, salah satunya adalah user information) - Profile Page
3. Multibahasa (Pilihan menggunakan bahasa yang bisa di translate by real time) - Home Page

**KESEHATAN**

4. Self-check gejala (AI sederhana berbasis input) (Reply bot yang reply nya sudah ditentukan dengan kata kunci yang di deteksi) - AI Page
5. Rekomendasi gaya hidup sehat (Pilihan dan rekomendasi gaya hidup sehat dan apa yang perlu dilakukan dan informasi kalori) - Healthy Page
6. Tracking kesehatan harian (makan, tidur, olahraga) (Informasi user mengenai ngetrack user sudah makan berapa kali, tidur berapa lama, dan olahraga menghabiskan berapa banyak kalori) - Healthy Page
7. Riwayat Chat dengan AI - AI Page
9. Screen Time Reminder - Universal


**OBAT & INFORMASI**

9. Database obat (Webpage dengan informasi berbagai macam obat dan kandungannya) - Medicine Page
10. Beli obat (Seperti online shop pada umumnya dan bisa melakukan checkout) - Shopping Page
11. Informasi penyakit umum (Webpage dengan informasi beberapa penyakit yang umum menyerang manusia) - Illness Page
12. Kalender (Menampilkan tanggal, daily reminder untuk tidur, makan, dan lain lain) - Home Page
13. Jenis jenis penyakit lanjutan dan gejalanya (Webpage dengan informasi beberapa penyakit lanjutan atau yang parah dan gejala awalnya (berbeda dengan penyakit yang umum menyerang)) - Disease Page


**LOKASI & AKSES**

14. Pencarian rumah sakit terdekat (Menampilkan beberapa informasi di kota tertentu yang dipilih oleh user dan yang dimiliki oleh partner bisnis kita) - Maps Page
15. Maps lokasi fasilitas kesehatan (Menggunakan API google maps untuk search fasilitas kesehatan) - Maps Page


**SISTEM**

16. Notifikasi (jadwal, obat, konsultasi) (Menampilkan notifikasi by kalender yang sudah mendekati jadwal reminder) - Home Page
17. Akses untuk memilih dokter untuk konsultasi offline (Memilih dokter untuk konsultasi offline/janji temu) - Doctor Page
18. Rating & review dokter (Lanjutan dari konsultasi offline atau emang mau rating dokternya secara online) - Doctor Page
19. Dokter yang tersedia dan tidak - Doctor Page
20. Informasi mengenai dokter (Informasi mengenai dokter beserta keahliannya dibidang apa) - Doctor Page
21. Toggle dark/light mode - Universal
22. Article kesehatan (Beberapa article mengenai kesehatan yang akan muncul di home page dan random) - Home Page
23. Masukan user (local storage) (Lanjutan dari rating dan review dokter untuk tetapi ini bisa untuk website dan segalanya) Rating Page

Minggu 4 : User & Account
Minggu 5 : Kesehatan, Obat & Informasi
Minggu 6 : Lokasi & Akses
Minggu 7 : Sistem

Header Dan Footer secara universal dan digunakan di semua page
